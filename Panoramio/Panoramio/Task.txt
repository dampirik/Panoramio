Используя Panoramio API (http://www.panoramio.com/api/data/api.html) создать приложение, 
которое будет показывать доступные фотографии вблизи указанной точки координат.
Пользователь указывает точку на карте;
Приложение выполняет запрос к Panoramio API и отображает список всех
доступных фотографий в указанной точке координат в режиме превью;
По тапу на превью открывается изображение в полном размере 
(по возможности можно предоставить возможность сохранить/расшарить фото)
Приложение должно быть универсальным (Universal App);
Обработка исключительных ситуаций и индикация выполнения запросов (в том числе загрузка изображений);
Творческий подход приветствуется.
